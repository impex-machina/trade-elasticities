# Supplementary materials

This directory holds post-submission additions to the methodology
evidence base — responses to reviewer comments, additional robustness
checks, and any analyses produced after the evidence documents in
`docs/methodology/` were frozen at paper submission.

The frozen evidence documents are not edited after submission; new
material lands here instead. See the "Provenance and updates" section of
`../README.md`.

This directory is a structural placeholder until such materials exist.
